Stock Price Trend Analysis and Prediction
A Mini Project for AI/ML Course (Google Colab + Python + Pandas + Matplotlib + Linear Regression)
Project Overview
This project performs data loading, cleaning, feature engineering, EDA (Exploratory Data Analysis), and prediction on stock-related datasets.
It includes two major parts:

1. Sales & Returns Analysis (Excel File with 3 Sheets)

File contains: Orders, People, Returns

DataFrames created: df_orders, df_people, df_returns

Key operations:

Loading multi-sheet Excel data

Exploring relationships between orders and returns

Joining datasets for deeper insights

Identifying which region/manager has the highest return rate
